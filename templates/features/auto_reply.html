强大的底层错误分类与智能重试机制

出海营销最怕网络波动和接口报错。我们的回复引擎内置了专业的错误分类逻辑，实现业务的高送达率。

单一数据源的错误分类器

系统在 app/common/error_classifier.py 中定义了统一的错误处理方案。自动将捕获的 Telethon 网络报错、Telegram API 频控限制和超时报错进行统一归类，不重复开发冗余规则。

毫秒级重试策略与防闪崩

将错误划分为“可重试（如连接超时）”与“不可重试（如账号失效、被禁言）”。可重试的错误自动加入重试队列进行指数退避规避风控，避免频繁报错拖垮号池整体活跃状态。
